# serve locally
run `npx @11ty/eleventy --serve` from the /tglyn.ch directory

## in debug mode
run `DEBUG=Eleventy* npx @11ty/eleventy --serve` from the /tglyn.ch directory

# push to netlify
make a commit to master and push to github





# how to get set up
- install nvm
- install latest version of node
- run npm install
- then run the above
